Library Card Catalog App

Version 0.0.1

2016-09-25

Before you Start

  • Set set your user name. This is required. It can be any short string
  • Use the Preferences entry in the Main Menu

Manage Books

  • Open Main menu: Sweep right or touch icon in upper-right of screen
  • The Book menu item lists all books
  • Search and create new books from the home page
  • The app requires permission to use the camera (for scanning barcodes)

Scan Barcodes and Search

  • Start barcode scanner from Home page
  • Seach for a record with the Search for ISBN button
  • If a book with the ISBN is not found, you can create a new record

Enter Book Data

  • Start from New Book botton on Home page, or the New Book button on the Book List
  • Use Get Book Details button to fill in the Title, Author and Publisher from the online ISBN database
  • Use the Avilaibility tag to record the Location of the book and if it is available for loan

Synchronize with the Server

  • Use the Synchronize entry in the main menu
  • Download server records with the Download button
  • Upload local records with the Upload button
